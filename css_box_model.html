CSS盒模型详解

什么是CSS盒模型？

CSS盒模型是网页布局的基础，每个HTML元素都被视为一个矩形盒子，由内容区（content）、内边距（padding）、边框（border）和外边距（margin）四部分组成。

盒模型组成部分

  • 内容区（Content）：显示元素的实际内容（文本、图片等），尺寸由width和height属性控制。
  • 内边距（Padding）：内容区与边框之间的空白区域，通过padding属性设置。
  • 边框（Border）：围绕内边距的线条，通过border属性设置宽度、样式和颜色。
  • 外边距（Margin）：边框与其他元素之间的空白区域，通过margin属性设置。

盒模型示例代码

.box {
    width: 200px;
    height: 150px;
    padding: 20px;
    border: 2px solid #3b82f6;
    margin: 10px;
    /* 标准盒模型（默认）：总宽度=width+padding*2+border*2 */
    box-sizing: content-box;
    /* 替代盒模型：总宽度=width（包含padding和border） */
    /* box-sizing: border-box; */
}

实际应用案例：03/04文档中的盒模型实践

03-css-01.html：样式表的盒模型应用 （文档名称：03-css-01.html，创建时间：2025-3-19）

该文档展示了三种样式表使用方式（外部/内部/内嵌），其中内嵌样式（如li元素的color属性）直接作用于元素盒模型的内容区样式；内部样式（h2的font-size等属性）定义了内容区的文字尺寸和装饰。

04-css-作业.html：container类的盒模型参数 （文档名称：04-css-作业.html，创建时间：2025-3-27）

.container类的CSS属性直接对应盒模型各部分：

  • max-width: 1200px - 控制内容区的最大宽度
  • margin: 0 auto - 外边距实现水平居中
  • padding: 30px - 内边距定义内容区与边框的空白
.container {
    max-width: 1200px; /* 内容区最大宽度 */
    margin: 0 auto; /* 外边距 */
    padding: 30px; /* 内边距 */
}
