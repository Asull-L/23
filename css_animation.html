CSS动画详解

什么是CSS动画？

CSS动画通过transition（过渡）和animation（关键帧动画）属性，为元素添加平滑的动态效果，无需JavaScript即可实现按钮悬停、菜单展开、加载提示等交互效果。

核心技术对比

  • transition：用于元素状态变化时的平滑过渡（如hover时颜色变化），需要触发条件（如:hover）。
  • animation：通过@keyframes定义关键帧，可自动循环播放，支持更复杂的动画序列（如旋转、淡入淡出）。

典型应用示例

<!-- 过渡动画示例 -->
.button {
    background: #3b82f6;
    color: white;
    padding: 8px 16px;
    transition: background 0.3s ease, transform 0.3s ease;
}
.button:hover {
    background: #2563eb;
    transform: scale(1.05);
}

<!-- 关键帧动画示例 -->
@keyframes fadeIn {
    from {
        opacity: 0;
        transform: translateY(20px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}
.animated-box {
    animation: fadeIn 0.5s ease-out forwards;
}

实际应用案例：07/08文档中的动画实践

07-async-01.html：JavaScript动画控制 （文档名称：07-async-01.html，创建时间：2025-4-15）

该文档展示了通过setTimeout、Promise和async/await控制CSS动画的执行时机，例如通过async函数延迟1秒后设置元素透明度。

// setTimeout 示例
setTimeout(() => {
    console.log('延迟2秒执行');
}, 2000);

// Promise 示例
function fetchData() {
    return new Promise((resolve) => {
        setTimeout(() => {
            resolve('数据加载完成');
        }, 1500);
    });
}
fetchData().then(data => console.log(data));

// async/await 示例
async function animateWithDelay() {
    await new Promise(resolve => setTimeout(resolve, 1000));
    document.querySelector('.animated-box').style.opacity = '1';
}
animateWithDelay();
