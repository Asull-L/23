响应式布局详解

什么是响应式布局？

响应式布局通过媒体查询（@media）、弹性布局（flex/grid）和百分比单位等技术，使页面内容能自动适配不同屏幕尺寸（手机、平板、桌面），提供一致的用户体验。

为什么需要响应式布局？

  • 多设备适配：覆盖手机、平板、桌面等不同终端，无需开发多个独立页面。
  • 提升体验：内容自动调整布局，避免用户手动缩放查看。
  • 优化SEO：单一URL对应所有设备，搜索引擎更易抓取。

常用实现技术

<!-- 视口元标签 -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<!-- 媒体查询示例 -->
@media (max-width: 768px) {
    .container {
        padding: 0 16px;
    }
}

<!-- 弹性布局示例 -->
<div class="flex flex-col md:flex-row gap-4">
    <div>左侧内容</div>
    <div>右侧内容</div>
</div>
